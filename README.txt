# Welcome to Guessing Games!

## Guessing Game 2 (`guessing_game2.py`)

### Overview
Guessing Game 2 is a simple number guessing game. The computer picks a random number between 1 and 100, and you have to guess it! The game will keep track of your scores and attempts and save them in a file called `game_results.txt`.

### How to Play
1. Open a terminal or command prompt.
2. Navigate to the folder containing `guessing_game2.py`.
3. Run the game by typing:
   ```bash
   python guessing_game2.py
4. Follow the on-screen instructions to input your guesses and enjoy the game!


Web Scraping Guessing Game is a fun challenge where you guess the title of a random Wikipedia article! The game fetches a random article using web scraping.
### How to Play
1. Open a terminal or command prompt.
2. Navigate to the folder containing webscraping_guessinggame.py.
3. Run the game by typing:
python webscraping_guessinggame.py

4. Follow the on-screen instructions to guess the title of the Wikipedia article.
